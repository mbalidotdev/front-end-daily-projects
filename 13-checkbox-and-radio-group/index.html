All radio buttons with the same name belong to the same group.

Key Differences: Radio vs Checkbox

Feature Radio Buttons Checkboxes
Selection Only ONE ZERO, ONE, or MANY
Deselect? Can't deselect (need another option) Can uncheck any time
Same name? YES - required for grouping Optional - same name = multiple values
Use case Gender, payment method, size Toppings, interests, agreement
